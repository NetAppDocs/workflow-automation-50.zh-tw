= 新增實體至WFA工作流程套件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以將一或多個實體新增OnCommand Workflow Automation 至位於WFA（WFA）的工作流程套件、以滿足儲存自動化和整合需求。

您可以從下列實體移除套件：

* 工作流程
* 尋找者
* 篩選器
* 命令
* 功能
* 範本
* 方案
* 字典
* 資料來源類型
* 遠端系統類型
* 快取查詢
* 類別


.步驟
. 透過網頁瀏覽器登入* WFA *視窗。
. 按一下「* Designer *>*」。
. 在「實體」視窗中、選取您要新增至套件的實體。
. 按一下「*新增至套件*」圖示。
+
「Add to Pack」（新增至套件）僅適用於認證設為「無」的實體。

. 在「*新增至套件*」對話方塊的「*可用套件*」下拉式清單中、選取您要新增實體的套件。
. 按一下「*確定*」。
